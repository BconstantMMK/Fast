FastLBM: FAST LBM structured grid Navier-Stokes solver

Preamble

FastLBM is an efficient Lattice Boltzman solver for use on structured grids.

FastLBM module works on CGNS/python trees (pyTrees) containing grid information (coordinates must be defined, boundary conditions and flow solution).

This module is only available for the pyTree interface:

import FastLBM.PyTree as FastLBM
