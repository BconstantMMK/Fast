FAST: Flexible Aerodynamic Solver Technology

FAST is a set of python modules implementing HPC solvers for Computational Fluid Dynamics (CFD).

FASTS is a very efficient solvers for compressible Navier-Stokes equations for Structured grids. It is well suited for unsteady applications (URANS, LES, ...)

FAST/IBM is a technology providing automatic mesh generation, using a very efficient Cartesian solver and immersed boundary conditions.

FAST modules are compatible with Cassiopee pre- and post-processing functions (http://elsa.onera.fr/Cassiopee).
