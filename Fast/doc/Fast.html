Fast: Flexible Aerodynamic Solver Technolgy

Version: 2.5 (20/09/2017)

Author: Onera

Preamble

Fast is a common/front module for all Fast series solvers.

Fast is only availble for use with the pyTree interface. You must import the module:
import Fast.PyTree as Fast
Then, in the following, a is a zone node and A is a list of zone nodes or a complete pyTree.

Actions

Fast.setNum2Base: set the numeric settings to bases. a is a pyTree or a base. numb is a dictionary with entries:
'temporal_scheme': 'explicit' or 'implicit' or 'implicit_local'
'ss_iteration': 30 (number of implicit iterations)
'modulo_verif': 20 (period of cfl extraction, newton convergence and predictor estimation):
A = Fast.setNum2Base(A, numb)
Example of use: set numeric settings in bases.

Fast.setNum2Zones: set the numeric settings to zones. numz is a dictionary with entries:
'scheme': 'ausmpred' or 'senseur' or 'roe_min'
'motion': 'none', or 'rigid' or 'deformation'
'omp': 0
'time_step': value of time step
'time_step_nature': 'global' or 'local'
'epsi_newton': 1.e-1 (newton stopping criteria)
'psiroe': 1.e-1 (Harten correction)
'cfl': usefull only for time_step_nature=local
'model': 'Euler' or 'NSLaminar' or 'NSTurbulent'(spallart)
A = Fast.setNum2Zones(A, numz)
Example of use: set numeric settings in zones.

Fast.metric: compute the metric needed by the solvers:
A, metrics = Fast.metric(A)
Example of use: compute the metrics (pyTree).

Fast.save: save computation tree t in file. If you run in mpi, NP must be the number of processor. If you run in seq mode, NP must be 0 or a negative number. If split='single', a single file is written. If split='multiple', different files are created depending on the proc number of each zone (restart/restart_0.cgns, ...):
Fast.save(t, fileName='restart.cgns', split='single', temporal_scheme='implicit', NP=0)
Example of use: save computation tree (pyTree).

Fast.load: load computation tree t from file. Optionaly load tc (connectivity file) or tstat (statistics file). Returns also the graph as a disctionary {'procDict', 'graphID', 'graphIBC'}. The meaning of NP is the same as for save:
t, tc, ts, graph = Fast.load(fileName='restart.cgns', fileNameC= 'tc.cgns', fileNameS='tstat.cgns', split='single', restart=False, NP=0)
Example of use: load computation tree (pyTree).
